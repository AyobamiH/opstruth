## Boundaries

- v0.1.1 is a public testing release, not a maturity claim.
- Cloudflare Workers is the production website target.
- Lovable remains a frontend editing surface, not production deployment.
- GitHub Actions deploys production website changes to Cloudflare.
- Cloudflare and npm secrets are stored in GitHub/npm/Cloudflare flows, not in code.
